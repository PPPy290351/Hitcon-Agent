# Vue and Socket.io

Demo app using Vue and Socket.io

- Runs port `8080`
- It communicates trough web sockets on port `5000`

# Running

## Vue app

```bash
$ cd web

# Install dependencies
$ npm install

# Serve with hot reload at localhost:8080
$ npm run dev
```

## Server

```bash
$ cd server

# Install dependencies
$ npm install

# Serve on localhost:5000
$ node server.js
```

## KofServer

```bash
$ cd kofserverMock

# Install dependencies
$ npm install

# Serve on localhost:9090
$ node server.js
```
